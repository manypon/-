# Яндекс Карты Парсер (Yandex Maps Scanner)

> **Статус: активная разработка**  
> Проект в стадии бета-тестирования. Базовый функционал работает, ведутся работы по улучшению стабильности и удалению дублей.

Простая программа для сбора данных об организациях с Яндекс Карт.  

## 🎯 Возможности

- Поиск по запросу и городу
- Сбор: адрес, телефон, сайт, рейтинг, соцсети (VK, Telegram, WhatsApp и др.)
- Сохранение в `.xlsx` (Excel)
- Поддержка Яндекс Браузера (и Chromium-браузеров)

## 🚀 Как использовать

1. Скачайте последнюю папку: MapParser - .py version
2. Установите необходимые зависимости:
[a] pip install customtkinter
[b] pip install playwright
[c] pip install pandas
[d] pip install openpyxl
3. Запустите папку main.py использовав Python

## 🚀 Как собрать .exe файл

1. Установите необходимую зависимость:
[a] pip install pyinstaller
2. Используйте библиотеку pyinstaller из файла main.py:
[b] pyinstaller --onefile --windowed --name="YandexMapsScanner" main.py
3. Запустите .exe файл из корневой папки в которой так-же присуствует папка _internal (она создается после сборки автоматически)

## 📁 Выходной файл

- Формат: `.xlsx` (Excel)
- Колонки: Название, Адрес, Телефон, Сайт, Рейтинг, Отзывы, VK, и др.
